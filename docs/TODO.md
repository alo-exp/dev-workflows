## Features

### Skill Gap Check — Skill Portals
When performing the skill gap check (post-DISCUSS and post-PLAN), skills will not only be
searched for in the local skill inventory. A curated list of **skill portals** will be
maintained — registries or repositories where useful skills can be discovered. When the gap
check identifies a capability not covered locally, the relevant portals will be searched
automatically. If a matching skill is found, the user will be asked whether to install it
before proceeding. Installed skills are then used in subsequent workflow steps.

**Portals to consider**: Claude Code plugin marketplace, GitHub topics (claude-code-skill,
claude-skill), curated community registries.
